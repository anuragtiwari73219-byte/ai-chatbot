# AI Chatbot 🤖

This is an LLM-based chatbot built using Python and LangChain.

* Handles user queries
* Generates intelligent responses
* Uses prompt engineering
* API-based responses

## Tech Stack

* LangChain
* OpenAI API

## How to Run

1. Install dependencies
2. Add API key
3. Run the script
